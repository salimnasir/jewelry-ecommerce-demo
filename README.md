# 💎 Jewelry E-commerce Website Demo

A modern and responsive jewelry e-commerce homepage inspired by premium brands like Pandora and Tiffany & Co.

---

## 🚀 Live Demo
https://salimnasir.github.io/jewelry-ecommerce-demo

## ✨ Features

- Fully responsive (mobile + desktop)  
- Product categories:

  - Necklaces

  - Earrings
